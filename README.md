# Test Cases Generator

## Project Overview

This project aims to automate the generation of test cases. More details will be added here as the project develops.

## Features

-   Automated test case generation based on Jira Story Description + Images

## Installation

To install the necessary dependencies, run:

```bash
pip install -r requirements.txt
```

## Usage

1. Provide JIRA related configurations in the .env file
2. Provide CLAUDE_API_KEY in the .env file
3. Run the script

```bash
python main.py
```

--> Provide the ticket id when requested in the terminal

## Configuration

Currently supporting claude code api
